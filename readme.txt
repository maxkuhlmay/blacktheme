== Beschreibung ==
Ein minimalistisches, responsives WordPress-Theme für Blogs und Websites.

== Installation ==
1. Lade das Theme-Archiv herunter.
2. Entpacke die Dateien und lade sie in das Verzeichnis `/wp-content/themes/`.
3. Gehe zu **Design -> Themes** und aktiviere das Theme.
4. Passe das Theme unter **Design -> Customizer** nach deinen Wünschen an.

== Features ==
* Anpassbare Farben
* Responsives Design
* Unterstützung für Widgets und Menüs
